# Featured Projects

## 🏥 PRIMS – Patient Record Interactive Management System

A web-based clinic management system developed as a year-long capstone project for Asia Pacific College's clinic.

Developed collaboratively by a team of five members, the system streamlines clinic operations through appointment scheduling, digital medical records, inventory tracking, and reporting.

### My Contributions

- Established the initial frontend structure and reusable layouts that served as the foundation for the application's user interface.
- Developed the landing page and the appointment history page for students, faculty, and non-clinic staff.
- Contributed to both the frontend and backend of the appointment scheduling module by implementing feature enhancements, bug fixes, and improvements throughout development.
- Developed the clinic staff scheduling module, allowing nurses to manage doctors' available dates and time slots while viewing and managing scheduled patient appointments by day.
- Designed and implemented the inventory management module, including medicine and stock management, automated low-stock and expiration alerts, medicine dispensing, and automatic inventory updates for auditing purposes.
- Contributed to the medical records module by improving the user interface, refining form validation, and implementing enhancements to existing functionality.
- Assisted in developing the summary reports module by integrating Google's Gemini API to generate AI-assisted insights and recommendations based on clinic data trends, helping identify patterns such as seasonal illnesses and expected healthcare demands.
- Designed and implemented a significant portion of the application's database interactions by developing models, controllers, and database logic to support the system's core features.
- Participated in client meetings with the APC clinic to gather and validate requirements, collaborated on system integration and testing, and presented the completed system to faculty and project stakeholders.

### Tech Stack

Laravel • Livewire • PHP • MySQL • Tailwind CSS
